# 🧴 Mizucare — AI Skin Analysis & Care Routine Platform

**Full-Stack Developer (Vibe-Coding MVP)**
Университетский экзаменационный проект: веб-приложение для анализа кожи лица с помощью искусственного интеллекта, подбора персонального ухода и косметических продуктов для девушек.

🔗 **Link:** [mizu-care-v2.vercel.app](https://mizu-care-v2.vercel.app/) | 💻 **GitHub:** [github.com/DefiveninthLabs/mizu-care](https://github.com/DefiveninthLabs/mizu-care)

---

### 🛠 Технологии
* **Frontend:** React, Next.js, Tailwind CSS
* **AI Интеграция:** Gemini API (мультимодальный анализ изображений и генерация структурированного JSON)
* **Автоматизация & Скрипты:** Python (парсер интернет-магазинов для наполнения базы данных)

---

### 🎯 Реализованный функционал
* **AI-Сканирование лица:** Интеграция Gemini API для обработки загруженных фотографий кожи лица совместно с результатами мини-опроса. ИИ анализирует состояние кожи и возвращает структурированный ответ.
* **Парсинг каталога косметики:** Написан Python-скрипт с использованием ИИ, который спарсил интернет-магазины косметики и автоматически заполнил базу данных 443 продуктами.
* **Персонализированный рутинный уход:** Динамический рендеринг JSON-массива от AI с пошаговыми рекомендациями по уходу и автоматическим подбором подходящих уходовых средств из собственной базы данных.
* **Мобильная адаптация:** Полностью отзывчивый интерфейс, оптимизированный под экраны мобильных устройств.
