# 👥 Электронная очередь Narxoz University
> **Frontend-разработчик** | Масштабируемое веб-приложение для цифровизации физических очередей в вузе.

Проект разработан с целью устранения хаоса, оптимизации потока абитуриентов и студентов во время приемной комиссии, а также предотвращения конфликтов в живых очередях за счет прозрачной цифровой системы.

---

## 🛠 Технологический стек и решения

* **Frontend:** React, Next.js, HTML5, CSS3 / Tailwind CSS (адаптивная верстка).
* **Real-time:** **WebSockets (Socket.io)** — интеграция для мгновенного обновления статуса очередей, количества человек в режиме реального времени и вызова к окну без необходимости перезагружать интерфейс.
* **API Integration:** Архитектура взаимодействия с бэкендом через RESTful API с обработкой состояний загрузки, ошибок и валидации данных пользователей.
* **Оптимизация:** Тонкая настройка рендеринга компонентов для обеспечения стабильной работы приложения при высоких нагрузках (в пиковые дни работы приемной комиссии).

## 💻 Интерфейс и функционал (Скриншоты)

### 1. Страница авторизации / Ввода персональных данных
Пользователь проходит первичную верификацию по имени и номеру телефона, выбирает язык интерфейса и дает согласие на обработку данных для постановки в очередь. Реализована маска ввода номера и валидация полей.

## 🎯 Мой вклад в проект

1. **Разработка UI-компонентов:** Полностью спроектировал и сверстал адаптивные экраны авторизации, выбора категорий и интерактивных карточек согласно дизайн-макетам.
2. **Связка с Websockets:** Реализовал подписку на события изменения состояния очереди. Когда оператор вызывает следующего человека, счетчик `People in line: X` мгновенно обновляется у всех пользователей в этой категории.
3. **Оптимизация производительности:** Использовал мемоизацию и оптимизировал подписки на сокеты, чтобы минимизировать лишние перерендеры страниц, когда сотни пользователей одновременно держат вкладку открытой.
4. **Интернационализация (i18n):** Настроил переключение языковых локалей (Казахский, Русский, Английский) на клиенте для удобства иностранных абитуриентов.
